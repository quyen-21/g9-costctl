# sample_output/

These files are the **real outputs** generated directly from running the `costctl` commands against the configured AWS account.

## Included Outputs

- `list_ec2.txt`: Real output for EC2 instances with tags.
- `list_ec2_missing_app.txt`: Real list of EC2 instances lacking the `Application` tag.
- `missing_tag_volume.txt`: Real list of EBS volumes lacking the `Application` tag.
- `cost.txt`: Real aggregated cost report retrieved via the AWS Cost Explorer.
- `idle.txt`: Real idle EC2 instances list scanned based on CloudWatch CPUUtilization.
- `migrate_gp3.txt`: Real dry-run migration plan showing gp2 volumes eligible for gp3.
- `terminate.txt`: Real output demonstrating the termination execution.

All of these outputs follow the exact columnar structure, padding, and layout generated by the CLI implementation.
